QUICKDRAW

A simple Python implementation of a QuickDraw drawing recognition project using machine learning and computer vision. Inspired by Google’s Quick, Draw! neural network drawing game.

🧠 About the Project

This repository contains code to build an interactive application that recognizes hand-drawn sketches using a trained CNN model. The system loads sketch data, preprocesses it, trains a classifier, and allows user drawings to be predicted in real time using webcam input.

Quick, Draw! is a web-based game created by Google where AI attempts to guess what a user is drawing in limited time. This project replicates similar functionality using Python.

🚀 Features

✔ Load and preprocess QuickDraw sketch data
✔ Train a Convolutional Neural Network (CNN) for sketch classification
✔ Predict hand-drawn doodles using webcam or input images
✔ Save and load trained models
✔ Easy setup and extensible design
